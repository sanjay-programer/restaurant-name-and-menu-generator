# 🍽️ AI Restaurant Name & Menu Generator

An AI-powered application that generates **creative restaurant names and complete menus** based on a selected cuisine.

This project was built as my **first AI-integrated application**, combining modern AI tools with a simple user interface to automate restaurant concept creation.

---

## 🚀 Features

- Generate **unique restaurant names**
- Automatically create **full menu ideas**
- Support for multiple **cuisine types** (Indian, Chinese, Russian, etc.)
- AI-generated **creative dish lists**
- Simple and interactive **web interface**

---

## 🧠 Tech Stack

**Frontend**
- Streamlit

**AI & Backend**
- Google Gemini
- LangChain

**Programming Language**
- Python

---

## ⚙️ How It Works

1. User selects a **cuisine type**.
2. The application sends the prompt to the **Gemini AI model**.
3. Using **LangChain**, the AI generates:
   - A restaurant name
   - A themed menu
4. The results are displayed instantly through the **Streamlit interface**.
